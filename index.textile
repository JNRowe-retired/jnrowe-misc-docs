h2. Introduction

Packages that don't naturally fit in to my other themed overlays.

If you find any problems with ebuilds in this overlay either file an "issue":http://github.com/JNRowe/misc-overlay/issues or drop me an "email":jnrowe@gmail.com.

h2. Installation

Up to date installation instructions are contained within the repositories @README.rst@ file, which can be "viewed online":http://github.com/JNRowe/misc-overlay/blob/master/README.rst.
